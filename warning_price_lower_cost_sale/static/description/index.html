On retail store customers can do negotiate the product that they want. But sometimes a cashier can do some mistake such change the product price lower than the product cost.

Using this module we can give an alert like red color on the product order line if cashiers do that mistake.


Here is the the result in the Sales Order form.

The cost of the item is IDR 60.000.
The entered unit price on the first row is IDR 50.000 so the row is displayed in red.
On the same product the unit price of the second line is IDR 62.000 so it is displayed in the normal color.
